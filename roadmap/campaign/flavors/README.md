# Campaign flavors — per-flavor authoring notes

Authoring bucket, not a feature thread. One file per house flavor
(Corporate / Feudal / Underworld / Sectarian), each holding the
nameset, mission-archetype catalog, and prose templates for that
flavor. The flavors themselves and their tone are designed in
[`../themes.md`](../themes.md); this directory is where the concrete
content for each lands as it's written.

Empty until the first flavor's content is authored.
